# 📝 C++ Todo App (Console)

A simple console-based Todo application built in C++.

## Features
- Add tasks
- View tasks
- Edit tasks
- Remove tasks
- Mark tasks as done
- Save/load from file

## How to run
1. Compile:
   g++ main.cpp -o todo
2. Run:
   ./todo

## Notes
- Data is stored in tasks.csv
